Что это?

Перед вами система учета посещаемости занятий в группе №132 факультета ПМ-ПУ (СПбГУ).

Используемые технологии

При разработке системы использовался полный стек JavaScript технологий. За серверную часть отвечает Socket.IO в связке с базой данных MongoDB.

Клиентская часть реализована в виде одностроничного веб-приложения построенного на AngularJS. Верстка и дизайн - Twitter Bootstrap
